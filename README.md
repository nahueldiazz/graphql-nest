# Dev
1. Clonar el proyect
2. copiar el ```env.template``` y renombrar a ```.env```
3. Ejecutar 
```
yarn install
```
4. Levantar la imagent (Docker desktop)
```
docker-compose up -d
``` 
5. levantar el backend de Nest
```
yarn start:dev
```
6. visitar el sitio ```localhost:3000/graphql```
